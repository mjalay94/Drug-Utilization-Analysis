Medicaid Drug Utilization & Spending Analysis  |  Tools: SQL (Access), Excel  |  GitHub
- Queried CMS State Drug Utilization 2023 dataset (data.gov) using SQL in Microsoft 
  Access to identify top prescribed drugs, state-level spending patterns, and quarterly 
  reimbursement trends across U.S. Medicaid programs.
- Built a multi-chart Excel dashboard with PivotTables, slicers, and scatter plot analysis 
  revealing cost-per-prescription disparities across 50 states, supporting data-driven 
  Medicaid spend insights.
